联系渠道
================

Web
---

- `pytest 问题跟踪器`_ 用于报告错误或建议功能。
- GitHub 上的 `pytest 讨论`_ 用于一般问题。
- `stackoverflow.com 上的 pytest <http://stackoverflow.com/search?q=pytest>`_
  用于发布带有 ``pytest`` 标签的精确问题。新问题通常会
  很快被 pytest 用户或开发者看到并回答。

Chat
----

- `pytest Discord 服务器 <https://discord.com/invite/pytest-dev>`_
  用于 pytest 开发可见性和一般帮助。
- ``#pytest`` `在 irc.libera.chat <ircs://irc.libera.chat:6697/#pytest>`_ IRC
  频道用于随机问题（使用 IRC 客户端，或 `通过网页聊天
  <https://web.libera.chat/#pytest>`_）
- ``#pytest`` `在 Matrix <https://matrix.to/#/#pytest:matrix.org>`_。

微博
-------------

- Bluesky: `@pytest.org <https://bsky.app/profile/pytest.org>`_
- Mastodon: `@pytest@fosstodon.org <https://fosstodon.org/@pytest>`_
- Twitter/X: `@pytestdotorg <https://x.com/pytestdotorg>`_
